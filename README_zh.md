# 每日简报仓库 📰

**自动化每日简报，包含天气、GitHub 趋势和开发见解。**

---

## 📋 关于本仓库

本仓库包含自动生成的每日简报，内容包括：

1. **🌤️ 天气报告** - 中国杭州的当前天气和预报，附带穿搭建议
2. **🔥 GitHub 热门仓库** - 近 30 天 Star 量最高的前 10 个仓库
3. **💡 创意开发建议** - 15-18 个创新的软件开发点子
4. **📌 总结与行动项** - 关键亮点和推荐行动

---

## 📁 目录结构

```
openClawBlog/

    ├── briefing_2026-03-12.md      # 英文版
    └── briefing_2026-03-12_zh.md   # 中文版
```

## 📄 文件命名规范

| 语言 | 格式 | 示例 |
|------|------|------|
| 🇬🇧 英文 | `briefing_YYYY-MM-DD.md` | `briefing_2026-03-12.md` |
| 🇨🇳 中文 | `briefing_YYYY-MM-DD_zh.md` | `briefing_2026-03-12_zh.md` |

每份简报文件开头都包含指向另一种语言版本的交叉链接。

## ⏰ 自动化时间表

| 任务 | 时间 (北京时间) | 描述 |
|------|----------------|------|
| **每日简报生成** | 上午 10:00 | 生成并推送双语简报 |
| **检查与同步** | 下午 2:00 | 验证生成和推送状态，必要时补充生成 |

## 🛠️ 技术栈

- **天气 API**: weatherapi.com
- **GitHub 数据**: GitHub Search API
- **自动化平台**: OpenClaw (AI 助手平台)
- **编程语言**: Node.js, Python, Bash

## 📊 简报内容详情

### 1. 天气与穿搭指南
- 当前温度、天气状况、湿度、风速
- 3-7 天天气预报
- 分层穿搭建议
- 降雨/紫外线提示

### 2. GitHub TOP 10 热门仓库
- 按 Star 数量排序（近 30 天）
- 详细描述和功能介绍
- 仓库直链
- 编程语言和标签信息

### 3. 创意开发建议
- AI 驱动的开发工具
- 工作流自动化
- 开发者体验优化
- 数据分析
- 安全与合规
- 新兴技术

### 4. 总结与行动项
- 每日亮点
- 推荐探索的仓库
- 明日预览
